**Systemless Magisk module that auto whitelists all user apps in Realme UI App Cloner. No more manual selection!**

---

## ✨ Features

- 🔄 **Auto Whitelist** — All user apps automatically added to App Cloner
- 🚀 **Boot Service** — Runs automatically on every reboot
- 🔍 **Smart Detection** — Detects new app installs and updates whitelist
- ⚡ **Instant Apply** — Force stops App Cloner to apply changes immediately
- 📦 **Preset Apps** — Essential apps like Via Browser always included
- 🛡️ **Systemless** — No system partition modification, fully reversible
- 🎯 **Minimal** — Clean code, no bloat, no unnecessary features
- 🌐 **Open Source** — MIT Licensed, free forever

---

## 📋 Requirements

- Realme device (ColorOS / Realme UI)
- Android 10 (Realme UI 1.0)
- Magisk 20.4+
- App Cloner must be enabled
